Bootstrap Tutorial

What is bootstrap?
Usually, we draw samples from the population and find out the sampling distribution of the statistic to make confidence intervals and carry out hypothese tests. But what if we dont know the data generator? We create a bootstrap population from the sample. We know the bootstrap population so we can figure out the bootstrap sampling distribution of the statistic.
Let y = (-4, 3, 5, 7) be the sample. The sample mean = (-4 + 3 + 5+ 7)/4 = 2.75
Now we take a bootstrap sample of size 4 with replacement from the bootstrap population. There are 4^4 = 256 distinct bootstrap samples.(You can consider each sample as a vector, so (-4, 3, 3, 3) is different from (3, -4, 3, 3))
