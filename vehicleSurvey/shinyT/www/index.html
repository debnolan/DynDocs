Bootstrap Tutorial

What is bootstrap?


Usually, we draw samples from the population and find out the sampling distribution of the statistic $\hat{θ}$. Then we can make confidence intervals and carry out hypothese tests. But what if we don't know the data generator? We can approximate the sampling distribution of $\hat{θ}$ with bootstrap. We use the sample as the bootstrap population so we can figure out the bootstrap sampling distribution of the statistic. We could simulate all $n^n$
Let y = (-4, 3, 5, 7) with n = 4 be the sample. The sample mean is 2.75, standard error is 4.79
We can use the bootstrap distribution of $y^*$ to estimate the standard error of y. $y^*$ is an observation drawn from the sample with $$ \Bbb {E}^*(y^*)= (-4)\Bbb {P}^*(-4) + (3)\Bbb {P}^*(3) + (5)\Bbb {P}^*(5)+ (7)\Bbb {P}^*(7) = {-4+3+5+7 \over 4} = 2.75 = \bar{y}$$ $$ Var^*(y^*) = \sum_{i={y^*}}(y^* - \Bbb {E}^*(y^*))^2\Bbb {P}^*(y^*) = {(-4 - 2.75)^2+ (3 - 2.75)^2+(5 - 2.75)^2+(7 - 2.75)^2 \over 4} = 17.1875 = {3\over 4} S_y^2 $$ $Var^*(y^*) = {n-1\over n}S_y^2 $ in general. Now we take a bootstrap sample of size 4 with replacement from the bootstrap population: $(y_{b1}^*, y_{b2}^*, y_{b3}^*, y_{b4}^*)$
(You can consider each sample as a vector, so (-4, 3, 3, 3) is different from (3, -4, 3, 3))
For each bootstrap sample we compute the bootstrap statistic $ \bar {y}_b^* = {y_{b1}^* + y_{b2}^* + y_{b3}^* + y_{b4}^* \over 4}$ We can find the sampling distribution of $ \bar {y}^* $ because we know $ \Bbb {P}(y_b^*) = {1 \over 256}$, because there are $4^4$ = 256 distinct bootstrap samples. Having the bootstrap distribution of $ \bar {y}^*$ , we can find $ \Bbb {E} (\bar {y}^*) $ and Var$(\bar {y}^*)$.
But also because $\bar {y}^*$ is linear and we know $ \Bbb {E} (\bar {y}^*) = \bar {y} $, and the $y_i^*$ are independent, we can determine
$$\Bbb {E}^*(y^*) = {1 \over n} \sum_{i=1}^n \Bbb {E}(y_i^*) = {1 \over n} \sum_{i=1}^n \bar {y} = \bar {y}$$
$$ Var^*(\bar {y}^*) = Var^*({1 \over n} \sum_{i = 1}^n y_i ^*)= {1 \over n} Var^*(y_i^*)= {1 \over n}({1 \over n}\sum_{i=1}^n(y_i - \bar {y})^2) = {1 \over n}({n-1 \over n}S_y^2)$$
For this example, we get $SD^*(\bar {y}^*) = 2.073 $






 Bootstrap Estimation 


More generally, without relying on the linearity of $ \hat {θ}$, 
$\Bbb {p}^* (y_1^*, ... , y_n^*) = (y_{b1}^*, ..., y_{bn}^*) = {1 \over n^n}$ let $\hat {theta}_b^* $be the statistic basted on an particular bootstrap sample(y_{b1}^*, ..., y_{bn}^*), we can compute
$$\Bbb {E}^*( \hat {θ}^*) = {1 \over n^n}\sum_{b=1}^{n^n} \hat {Θ}(y_{b1}^*, ..., y_{bn}^*)$$
$$Var^*(\hat {θ}^*) = {1 \over n^n}\sum_{b=1}^{n^n}[\hat {Θ}(y_{b1}^*, ..., y_{bn}^*) - \Bbb {E}^*( \hat {θ}^*)]^2 $$
Often, $n^n$ is a very large number (e.g, $4^4$ = 256, $10^{10}$ = 10,000,000,000, and $100^{100}$ = 10 billion), so figuring out the exact bootstrap distribution quickly becomes unwieldy, 
even for reasonable sample sizes. Fortunately, we can take a random sample of bootstrap samples and can do quite well at estimating 
$\Bbb {E}^*( \hat {θ}^*) \approx {1 \over 1000}\sum_{b=1}^{1000} \hat {Θ}(y_{b1}^*, ..., y_{bn}^*)$ where $(y_{b1}^*, ..., y_{bn}^*)$ is a random selected bootstrap sample. Similarly 
$Var^*(\hat {θ}^*) \approx {1 \over 1000}\sum_{b=1}^{1000}(\hat {Θ}(y_{b1}^*, ..., y_{bn}^*) - \bar{ \hat {θ}}^*)^2 $.

Using the bootstrap estimates, we can find the bootstrap confidence interval for θ.


    
 
  • Normal Theroey
    
    Assume $\hat{θ}$ has an normal distribution and use the bootstrap to estimate SE($\hat{θ}$) $\hat{θ} \pm z_{α/2} SE^*(\hat{θ}^*)$
    • 

 
  • Percentile ($\hat{θ}_{α/2}, \hat{θ}_{1-α/2}$)
    
     For a bootstrap sample with size n, find the (1-α)% confidence interval by ordering $ \hat{θ}_b^* $ from smallest to largest and then pick 
     $\hat{θ}_{(αn/2)}$ as the lower bound and $\hat{θ}_{(n-αn/2)}$ as the higher bound.
    • 








 Why does this work? 



    

  1. Bootstrap sample is drawn from the bootstrap population in the same manna as the sample is drawn from the population, so the bootstrap observation are analogous to the original observations.
    2. 

  2. Bootstrap statistic is computed from the bootstrap sample in the same manna as original statistic is computed from sample, so the bootstrap statistic are analogous to the original statistic. 
    3. 

  3. $\hat{θ}$ as an estimate of θ is analogous to $\hat{θ}^*$ as an estimated of $\hat{θ}$. 
    4. 

  4. the distribution of $\hat{θ}^*$ is analogous to the distribution of $\hat{θ}$ (which we don't know because the population is unknown).
